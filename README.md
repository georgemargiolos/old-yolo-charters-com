# Old YOLO Charters Website Analysis

This repository contains an analysis of the front-end resources used by the yolo-charters.com website. The analysis was performed to identify all fonts, scripts, libraries, and CSS resources for the purpose of replicating the site's styles and functionality.

## Contents

- **website_analysis.md** - Comprehensive documentation of all detected technologies, libraries, and resources
- **resources_analysis.json** - Machine-readable JSON file containing extracted resources
- **pasted_content*.txt** - Original HTML page source files from various pages of the website
- **analyze_resources.py** - Python script used for initial resource extraction
- **comprehensive_analysis.py** - Python script used for comprehensive analysis

## Key Findings

### CSS Libraries
- jQuery UI
- Magnific Popup
- PhotoSwipe
- Select2

### JavaScript Libraries
- jQuery
- Litepicker (date picker)
- jQuery UI
- Select2 (enhanced dropdowns)
- PhotoSwipe (image gallery)
- Magnific Popup (lightbox/modals)
- jQuery Bootpag (pagination)
- Google reCAPTCHA

### CDN Resources
- Litepicker mobile plugin (jsdelivr)
- jQuery Bootpag (cdnjs)
- Google reCAPTCHA

## Usage

Review the `website_analysis.md` file for a detailed breakdown of all resources and their purposes.
